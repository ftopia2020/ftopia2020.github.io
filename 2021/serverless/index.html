安装 & Demo

安装 serverless & 查看版本

$ npm install -g serverless #npm 全局安装 serverless
$ serverless –v #查看 Serverless Framework 的版本信息

配置 & 安装依赖

$ serverless init fullstack  #使用serverless init命令，下载相关 template

“在项目模板中找到.env.example 文件，修改名称为.env” 实际没找到类似 .env 文件 …

反馈了腾讯云问题，不知何时会改

笔者使用 VSCode 打开了该项目文件夹，新建了 .env 文件，按文档要求加入配置信息

# .env
TENCENT_SECRET_ID=xxx  // 您账号的 SecretId
TENCENT_SECRET_KEY=xxx // 您账号的 SecretKey

# 地域可用区配置
REGION=ap-guangzhou //资源部署区，该项目中指云函数与静态页面部署区
ZONE=ap-guangzhou-2 //资源部署可用区 ，该项目中指 DB 部署所在的可用区


$ npm run bootstrap #通过执行以下命令，安装所需依赖

部署 & 查看 Demo

$ sls deploy --all

访问 Demo，哇哦~

查看 API 请求，为空，但是 API 服务 200 (^▽^)

看到有个 Form 可以 Submit，Just try it~ (～￣▽￣)～
