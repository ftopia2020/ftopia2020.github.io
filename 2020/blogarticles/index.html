技术文章分类参考

  By me ♥   2020-12-22   934 字   2 分钟

写这篇源于偶然看见掘金的一个活动： 掘力计划创作者训练营第一期，开营了！

以下为总结的技术博文分类，希望能更好地持续输出优质内容~ P.S 参与训练营之后会输出总结 💪

废话不多说，直接上图，分享链接

博文分类

列举文字版大纲如下，博文示例链接可点（部分来源掘金等平台，可能会失效）

技术细节

  • 侧重于解决某个具体的问题，包括但不限于以下：
    • 工作中遇到（开源库有）什么bug？如何解决这个bug？
    • 如何使用某个具体的技术细节？
  • 博文示例
    • 解决 go get 下载包失败的问题
    • 记一次不寻常的 input checkbox 样式调整优化
    • 纯 CSS3 实现打字/输入的效果
    • Golang 使用 string.split 切割字符串并返回数组
  • 博文特点
    • 目标/标题比较明确
    • 切入点小，通常一篇文章搞定且不太长

干货类型

  • 对某一块内容进行的总结，具有较高的收藏价值，包括但不限于以下
    • 专题研究
    • 经验之谈 / 职业之路
    • 读书/学习笔记
    • 工具/框架简介
  • 博文示例
    • 大厂都是怎么 SQL 调优的
    • IM消息系统的设计和实现
    • 图解 IP 基础知识
    • 前端学习路线
    • 打工四年总结的数据库知识点
  • 博文特点
    • 属于较为受欢迎的一类文章，也更加容易在社区内和社区外传播
    • 切入点比技术细节大多了，通常一篇文章讲不完，可切分为 上/中/下 或者系列文章
    • 较实践总结相比，代码出现没有那么多；较多为理论和经验输出

实践总结

  • 实践总结型的文章更像是技术细节型和干货型的结合，面向特定领域的技术，包括但不限于以下
    • 相关技术总结/集锦
    • 数据结构 / 算法分析等
    • 语言类教程（俗、但受欢迎），如 XXX 语言入门 / XXX 快速上手
    • 核心技术点的实践及方案
    • 开发/框架具体使用及实践
  • 博文示例
    • 8个硬核技巧带你迅速提升CSS技术
    • 【机器学习与数据科学之旅】1 Numpy 实例指南
    • 3 分钟搞定 Vue 2.0 服务端渲染
    • Android和iOS开发中的异步处理（一）——开篇
    • Android和iOS开发中的异步处理（二）——异步任务的回调
  • 博文特点
    • 通常依赖于代码，读者需要根据代码一步步往下深入
    • 内容通常也比较多，可切分为 上/中/下 或者系列文章

翻译文章

  • 好的外文文章翻译为中文也是具有很高的价值，需标识[译] 及原文出处
  • 博文示例
    • [译] 我最终是怎么玩转了 Vue 的作用域插槽
    • Android7.0写给开发者的一封信（官网同步翻译）

杂谈 & 面经

  • 不属于以上类型的其他类文章，比如业内趣闻等；特别把面经提出来，你懂的 😂
  • 博文示例
    • 百度为什么掉队了
    • Facebook面经记
